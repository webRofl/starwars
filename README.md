## Инструкция по запуску приложения **локально**

<br />

<br />

Стартовая страница приложения localhost:3000/#/people

<br />

_Приложение использует [Node.js][2] для своей работы_

```
$ git clone https://github.com/webRofl/starwars.git
$ cd starwars/
$ npm i
$ npm start
```
